→ Test with a verified script from the built-in Script Hub first. If hub scripts work but your custom script doesn't, the issue is script-side, not executor-side. Verify your script's compatibility with the current game version.

**❌ Custom UI theme not saving**
→ Go to **Settings > UI > Theme** and click **Save to Profile** manually. If themes still don't persist between sessions, ensure Zenith has write permissions to its installation directory.

**❌ Auto-update stuck on "Checking for updates"**
